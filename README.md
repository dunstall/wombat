## Build
Run:
```
./bin/bazelisk build ...
```

And to run unit and integration tests run:
```
./bin/bazelisk test ...
```

Note bazelisk is compiled for Linux x64. If needed install the correct binary
from [Bazelisk](https://github.com/bazelbuild/bazelisk).
